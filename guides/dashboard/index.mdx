The [Paxos Dashboard](https://dashboard.paxos.com/) enables [treasurers](/guides/dashboard/roles#treasurer) and other institutional users to securely [deposit](/guides/dashboard/deposit) and [transfer](/guides/dashboard/transfer) fiat, stablecoins and other assets.
[Sign up for an account](https://dashboard.paxos.com/).

If you already have an institutional account, see [here](/guides/dashboard/migrate-to-dashboard) how to migrate.
