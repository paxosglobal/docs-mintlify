---
title: 'Fiat Deposit Instructions'
description: Access saved fiat deposit instructions for receiving USD into your Paxos profile.
---

![Fiat Deposit Instructions](/images/fiat-deposit-instructions.png)

Use the **Deposit Instructions** tab in [Fiat Accounts](https://dashboard.paxos.com/dashboard/fiat-accounts) to access saved fiat deposit instructions. To download deposit instructions, select the **Actions** menu and use **Export Instructions**.

<Tip>

Deposit instructions are immutable. If your deposit requirements change, create a new set of instructions.

</Tip>
