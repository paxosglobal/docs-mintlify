title: "Data Dictionary"
description: "QuickFIX-compatible FIX 4.2 data dictionary for the Paxos FIX interface."

Use this data dictionary with QuickFIX, QuickFIX/J, QuickFIX/n, and compatible FIX engines. It defines only the messages, fields, and values supported by the Paxos FIX interface, enabling strict message validation.

Save the XML below as `FIX42-paxos.xml` and reference it in your FIX engine configuration.
