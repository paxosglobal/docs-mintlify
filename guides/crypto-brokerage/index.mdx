<Tabs>
  <Tab title="API">
    Get [market data](/api-reference/market-data), create [orders](/api-reference/orders), execute [held-rate quotes](/api-reference/quotes), and [transfer](/api-reference/transfers) assets with [Paxos API v2.0](/api-reference).
  </Tab>

  <Tab title="FIX">
    Leverage the [Paxos FIX API](https://developer.paxos.com/docs/itbit/fix/) to trade crypto.

    Learn more about [FIX](/guides/crypto-brokerage/fix) and check out the [onboarding](/guides/crypto-brokerage/fix/onboard) and [certification](/guides/crypto-brokerage/fix/certify) guides to get started.
  </Tab>

  <Tab title="WebSocket">
    Construct the state of the order book and stream executed orders in real-time using the [Paxos WebSocket API](/guides/crypto-brokerage/api/websocket).

    Learn more about what you can do with [WebSocket](/guides/crypto-brokerage/api/websocket) or check out the [quickstart](/guides/crypto-brokerage/websocket) for a hands-on introduction.
  </Tab>
</Tabs>
