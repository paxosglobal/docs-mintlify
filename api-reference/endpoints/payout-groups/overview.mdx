---
title: "Overview"
description: "A Payout Group is a collection of reward addresses that receive reward distributions (claims) collectively."
---

## Key Features

- Create payout groups to collectively distribute rewards across addresses
- Update group name and destination address as needed
- Delete payout groups that are no longer needed
- Payout groups integrate with claim schedules for automated distributions
