---
title: "Overview"
description: "A Payout Group is a collection of reward addresses that receive a proportional share of reward distributions."
---

Payout groups let you distribute reward payments across multiple addresses in a single configuration. Each group defines how rewards are split among its members, simplifying multi-address reward management.

## Key Features

- Create payout groups to distribute rewards across multiple addresses
- Update group membership and distribution rules as needed
- Delete payout groups that are no longer needed
- Payout groups integrate with claim schedules for automated distributions
