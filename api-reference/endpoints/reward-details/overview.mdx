Use the Rewards endpoints to retrieve detailed reward information for your organization, including earned amounts by reward type and the time period they cover.

## Key Features

- Retrieve reward details for any eligible time period
- View a breakdown of reward types (e.g., custody, mint, acceptance)
- Use reward details for reconciliation and reporting

## Availability

<Note>
  Rewards Engine is currently in Preview and available to selected partners. Endpoints and behavior are under active development and subject to change.
</Note>
