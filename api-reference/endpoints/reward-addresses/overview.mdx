---
title: "Overview"
description: "A Reward Address is a blockchain address registered to receive reward payouts for eligible stablecoin activity."
---

Reward addresses are the destination addresses where Paxos sends reward payments. You can register one or more reward addresses per profile to control where reward payouts are delivered.

## Key Features

- Register blockchain addresses to receive reward payments
- Manage and update reward destination addresses per profile
- Reward addresses are validated against supported networks before activation

## Availability

<Note>
  Rewards Engine is currently in **Preview** and available to selected partners. Endpoints and behavior are under active development and subject to change.
</Note>
