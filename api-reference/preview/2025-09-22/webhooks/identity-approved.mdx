Triggered when identity verification is successfully approved.

<Note>
Webhook payloads contain only event metadata. Use the [Get Event API](/api-reference/preview/2025-09-22/endpoints/events/get-event) with the event ID to retrieve full details.
</Note>

## Event Object Structure

The Get Event API returns an [`identity_status_change`](/api-reference/preview/2025-09-22/events/identity-status-change-object) object with status transition details.
